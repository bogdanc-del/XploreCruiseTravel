 7. SCRIPTS & CRON JOBS
================================================================================

 Script                         Command                          Schedule
 ────────────────────────────────────────────────────────────────────────────────
 Cruise Data Sync               npm run sync:cruises                 AUTOMATIC — GitHub Actions
   - GitHub Actions: .github/workflows/sync-cruises.yml
   - Schedule: Daily at 04:00 UTC (07:00 EET Romania)
   - Fetches updated prices, images, departures from croaziere.net API
   - Updates cruises.json, cruises-enriched.json, cruises-index.json
   - Commits and pushes → triggers Vercel auto-deploy
   - Can also be triggered manually via GitHub Actions UI
   - Logs to logs/sync-log.jsonl

 Route Map Generation           npm run generate-route-maps          After sync (auto-triggered)
   - Generates SVG route maps for cruises

 i18n Audit                     npm run i18n-audit                   Before each commit (in test suite)
   - Checks translation key coverage EN/RO

 BNR Exchange Rate              AUTOMATIC (no cron needed)
   - Fetched on-demand via /api/exchange-rate
   - Cached 4 hours server-side (in-memory)
   - Cached 1 hour on CDN (Cache-Control header)
   - Client fetches once on app mount via ExchangeRateContext
